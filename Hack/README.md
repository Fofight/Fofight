# Fofight

scapy库可以很方便地实现跨平台的网络嗅探、网络发包等需求

文档分析工具PyPDF提供了强大的PDF格式解析功能，这些功能对PDF的Fuzz测试、PDF 0day的分析，甚至PDF Exploit的编写都起了极大的帮助。

 Linux 环境下的磁盘和内存取证工具包括 Ftkimage、xmount、Volatility、dd、netcat 等
